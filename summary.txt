  Key architectural decisions:
  - IndexedStack nav — screens stay alive when switching tabs
  - StateNotifierProvider per feature — clean separation of state and UI
  - Derived providers (filteredCatalogProvider, ordersTabCountProvider, etc.) for computed state
  - All widgets are decoupled — screens only import providers and widget files, never other screens
